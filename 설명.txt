외부데이터를 활용한 시각화
 - 원본폴더내의 공공데이터자료 "키워드 노인 "
 - csv 파일 vscode에서 open 시 인코딩 euc-kr 혹은 CP949로 처리


'분석 관점(Analysis Perspective)' 또는 '분석 인사이트(Analysis Insights)
-뉴스 데이터 기반 시각화 주제 및 설계 10선

1. 주제 : **주요 언론사별 뉴스 발행량 비교**
    주력컬럼 : 언론사, 일자
    시각화 메서드 : 막대 차트 (Bar Chart)
    X축 : 언론사
    Y축 : 뉴스 발행 건수

2. 주제 : **시간 흐름에 따른 사회적 카테고리 변화 추이**
    주력컬럼 : 일자, 통합 분류1
    시각화 메서드 : 누적 영역 차트 (Stacked Area)
    X축 : 일자 (월/분기별)
    Y축 : 분류별 뉴스 비율 및 건수

3. 주제 : **지역 뉴스에 나타난 핵심 지역 분포**
    주력컬럼 : 통합 분류1 (지역명), 특성추출 (지명)
    시각화 메서드 : 트리맵 (Treemap)
    X축 : 지역 계층 (대분류 > 소분류)
    Y축 : 사각형 면적 (뉴스 빈도수)

4. 주제 : **새해 첫날(신년) 가장 많이 언급된 핵심 키워드**
    주력컬럼 : 키워드, 일자
    시각화 메서드 : 워드 클라우드 (Word Cloud)
    X축 : - (단어 배치)
    Y축 : 키워드 언급 빈도 (글자 크기)

5. 주제 : **뉴스 분류와 언론사 간의 연관성 분석**
    주력컬럼 : 언론사, 통합 분류1
    시각화 메서드 : 히트맵 (Heatmap)
    X축 : 언론사
    Y축 : 통합 분류1 (주제)

6. 주제 : **뉴스 본문 특성(인물/기관/장소) 추출 빈도**
    주력컬럼 : 특성추출, 통합 분류1
    시각화 메서드 : 가로 막대 차트 (Horizontal Bar)
    X축 : 특성 추출어 언급 빈도
    Y축 : 특성 추출어 (인명/지명 등)

7. 주제 : **시간대별 '복지/노인/이웃' 관련 뉴스 증감**
    주력컬럼 : 일자, 키워드 ('노인', '이웃')
    시각화 메서드 : 선 그래프 (Line Chart)
    X축 : 일자 (날짜별 흐름)
    Y축 : 복지 키워드 언급 건수

8. 주제 : **언론사별 뉴스 텍스트 데이터의 복잡도(길이) 분석**
    주력컬럼 : 언론사, 키워드 (개수), 특성추출 (개수)
    시각화 메서드 : 박스 플롯 (Box Plot)
    X축 : 언론사
    Y축 : 키워드/특성 추출 가짓수 분포

9. 주제 : **'신년 동정' 뉴스의 인물 및 직책 네트워크**
    주력컬럼 : 특성추출 (인물/직책), 제목 ('동정')
    시각화 메서드 : 네트워크 그래프 (Network Graph)
    X축 : 노드 (인물-기관 연관성)
    Y축 : 연결선 (함께 언급된 빈도)

10. 주제 : **계절/날씨 관련 기사와 사회적 이슈(사망/한파 등) 연계**
    주력컬럼 : 일자, 키워드 ('한파', '사망'), 통합 분류1
    시각화 메서드 : 산점도 (Scatter Plot)
    X축 : 일자 (날짜)
    Y축 : 해당 키워드 뉴스 발생 빈도
